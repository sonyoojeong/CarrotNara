## 🖥️ 프로젝트 소개
 중고거래 플랫폼 당근나라 사이트 입니다.
 <br>
 
## 🕰️ 개발 기간 
* 23.02.24 ~ 23.03.17 

### 🧑‍🤝‍🧑 맴버구성
📌 커뮤니티팀 : 정지웅, 손유정 
- 거래후기 게시판 (CRUD), 공지사항 게시판 , Q&A 게시판(CRUD) , 사기이력 조회(검색), CSS, 당근나라 모션그래픽 로고


📌 상품팀 
- 상품 등록, 상품 댓글, 예약하기, 좋아요(찜)


📌 멤버팀
- 회원가입, 회원정보 수정, 삭제, 로그인, 메인 페이지, 찜 목록
